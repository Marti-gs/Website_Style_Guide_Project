Text Styles

How texts should be displayed.

H1 Headings

  • Font weight: bold (700)
  • Font size: 30px
  • Font family: Work Sans
  • Text transform: uppercase

H2 Headings

  • Font weight: bold (700)
  • Font size: 26px
  • Font family: Work Sans
  • Text transform: none

H3 Headings

  • Font weight: regular (400)
  • Font size: 25px
  • Font family: Merriweather Sans
  • Text transform: underline

Paragraph

  • Font weight: regular (400)
  • Font size: 22px
  • Font family: Merriweather Sans
  • Text transform: none
